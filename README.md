#  Barbie Movies IMDb Analysis — Power BI & Python

Este projeto apresenta uma **análise exploratória e visual** das avaliações IMDb dos filmes da Barbie, utilizando **Python para análise de dados** e **Power BI para visualização interativa**.

O objetivo é demonstrar habilidades práticas em **Data Analytics**, **Business Intelligence**, **Storytelling com dados** e **visual design**, aplicadas a um dataset temático e criativo.

---

## Objetivos do Projeto

- Analisar a evolução das avaliações IMDb dos filmes da Barbie ao longo dos anos
- Comparar o desempenho médio dos filmes por **Era** (Clássica, Transição e Moderna)
- Identificar os **temas com maior média de avaliação**
- Criar um **dashboard interativo, elegante e intuitivo**, pronto para portfólio

---

## Análises Realizadas

###  Evolução das Avaliações IMDb por Ano
- Visualização da média das notas ao longo do tempo
- Comparação entre as diferentes **eras dos filmes**

### Média IMDb por Era
- Filmes **Clássicos** apresentam maior média geral
- Queda gradual durante a fase de **Transição**
- Estabilização na era **Moderna**

###  Top 10 Temas com Maior Média IMDb
- Identificação dos temas mais bem avaliados
- Ranking dinâmico com filtros interativos

###  KPIs Principais
- Total de filmes
- Média IMDb geral
- Nota máxima
- Nota mínima
